Etiquetas Básicas de HTML, atributos y valores

HTML este es un lenguaje de marcado estándar para crear páginas web y text markup languaje, describe la estructura de una pagina web, consta de una serie de elementos y estos elementos le dicen al navegador como mostrar el contenido. También estos elementos etiquetan partes de contenido como “esto es un encabezado”, “esto es un párrafo”, “esto es un enlace”, etc. El HTML tiene unos atributos y valores, los atributos proporcionan información adicional sobre los elementos HTML porque todos los elementos HTML pueden tener atributos, estos atributos proporcionan información adicional sobre los elementos, también siempre se especifican en la etiqueta de inicio y generalmente vienen en pares de nombre/valor como:

nombre =” valor”, existen atributos como:

El atributo href, atributo src, atributo de ancho y alto, atributo alt, atributo de estilo, atributo Lang, atributo de título.

Los elementos en HTML tienen atributos; estos son valores adicionales que configuran los elementos o ajustan su comportamiento de diversas formas para cumplir los criterios de los usuarios.
